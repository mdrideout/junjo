## Features

<CardGrid stagger>
  <Card title="Eval Driven Development" icon="approve-check-circle">
    Use pytest to test each workflow node against unlimited test inputs and outputs, with asserts and LLM powered evals.
  </Card>
  <Card title="OpenTelemetry Native" icon="seti:plan">
    Execution traces are grouped into workflows and nodes. Optional self-hostable [companion telemetry server](https://github.com/mdrideout/junjo-server).
  </Card>
  <Card title="State Management" icon="seti:json">
    Redux-inspired state management for your workflows. Use [Junjo Server](https://github.com/mdrideout/junjo-server) to visually step through workflow state
    changes.
  </Card>
  <Card title="Read the docs" icon="open-book">
    Check out the Python library API [the docs](https://python-api.junjo.ai/). Find [Junjo on PyPI](https://pypi.org/project/junjo/).
  </Card>
